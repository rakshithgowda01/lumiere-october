# Lumière

A Next.js 14 portfolio site for Lumière — cinematic editing, reels, and brand storytelling.

## Tech
- Next.js 14, React 18, TypeScript
- Tailwind CSS v4
- Framer Motion

## Getting Started
```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
pnpm start
```

## Environment
Create a `.env.local` if needed:
```
NEXT_PUBLIC_CAL_URL=https://cal.com/your-username/15min
```

## Deployment
- Push to GitHub then import to Vercel. Framework preset: Next.js.
- Env vars: set `NEXT_PUBLIC_CAL_URL` as needed.
